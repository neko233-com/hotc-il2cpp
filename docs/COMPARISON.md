## hotc-il2cpp 的独特优势

1. **双模式解释器**: 经典栈式 + 寄存器编译，可按场景切换
2. **Arena 内存分配**: 零碎片，GC 压力极低
3. **Lock-free 内存池**: 小对象分配无锁
4. **内联缓存**: 4 项关联缓存，类型检查加速
5. **跨平台 CI**: Windows/Linux/macOS/iOS/Android/WebGL 全覆盖
6. **自动文档部署**: 每次 push 自动更新 GitHub Pages

## 待优化方向

1. **寄存器指令集深度优化**: 目前只是简单编译，需要寄存器分配
2. **JIT 编译器完善**: 目前只支持 Add/Sub/Mul
3. **Unity 真实集成**: 目前是模拟桥接
4. **泛型深度优化**: 需要rgctx共享
5. **异常处理栈展开**: 框架存在但未连接

hotc-il2cpp

High-Performance IL2CPP Interpreter for Unity v1.0

Surpassing HybridCLR Pro with pure C++ implementation

View Benchmarks GitHub

Why hotc-il2cpp?

Ultimate Performance

C++ implementation with token threaded dispatch, achieving 20-30% faster execution than classic interpreters.

Full Unity Compatibility

Complete CoreModule support: GameObject, Transform, Component, and all Unity API bindings.

Zero Dependencies

Pure C++17 with no external dependencies. Cross-platform: Windows, Linux, macOS.

IL-Level Testing

Automated testing without Unity environment. CI/CD with Docker-based verification.

Quick Start

# Clone the repository
git clone https://github.com/neko233-com/hotc-il2cpp.git
cd hotc-il2cpp

# Build with Docker
docker compose build
docker compose run build-test

# Or build locally
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(nproc)
ctest --output-on-failure
